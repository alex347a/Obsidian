#### Bits and bytes
Bit is a unit of information a byte is 8 bits.
on or off is 1 bits. 
4 states are 2 bits
numbers are more bits.

#### Communicating one bit
Examples are
Wire (copper):

| 0    | 1     |
| ---- | ----- |
| 0 V  | 5 V   |
| 12 V | -12 V |
Radio (air)

| AM                |
| ----------------- |
| FM                |
| frequency-hopping |
AM stands for amplitude-modulation
FM stands for frequency-modulation

Light (fiber/air)

| 0         | 1       |
| --------- | ------- |
| Color     |         |
| No light  | Light   |
| Flag down | Flag up |
